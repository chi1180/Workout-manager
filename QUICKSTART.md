# クイックスタートガイド 🚀

このガイドでは、ワークアウトマネージャーをセットアップして動かすまでの手順を説明します。

## 📋 前提条件

- Node.js 18以上がインストールされていること
- npm または yarn がインストールされていること

## ⚡ 3ステップでスタート

### 1️⃣ 依存関係のインストール

```bash
cd workout-manager/app
npm install
```

### 2️⃣ 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開きます。

### 3️⃣ プロダクションビルド

```bash
npm run build
```

ビルドされたファイルは `dist/index.html` に出力されます。

## 🎯 初回起動時の流れ

1. **オンボーディング画面**が自動的に表示されます
2. 6つの質問に答えてください：
   - 現在の運動習慣
   - トレーニングの目的
   - 1日に使える時間
   - 利用可能な器具
   - 避けたい動き
   - 好みの時間帯
3. 回答に基づいて**自動的にトレーニングプランが生成**されます
4. ダッシュボードで今日のエクササイズを開始！

## 📱 使い方

### ダッシュボード
- 今日のエクササイズが表示されます
- 各エクササイズのチェックボックスをタップして完了マーク
- 全て完了すると祝福アニメーションが表示されます！

### 履歴
- 年間のトレーニング履歴をヒートマップで確認
- 現在のストリーク（連続記録）を確認
- 月別の詳細記録を閲覧

### 設定
- プロファイルの再設定
- ダークモード切り替え
- データのバックアップ（エクスポート/インポート）
- 全データのリセット

## 🛠️ 開発時のコマンド

```bash
# 開発サーバー起動（ホットリロード有効）
npm run dev

# プロダクションビルド
npm run build

# ビルドしたファイルをプレビュー
npm run preview

# リンターを実行
npm run lint
```

## 📦 単一HTMLファイルについて

ビルド後、`dist/index.html` には以下が全てインライン化されています：
- React アプリケーション本体
- Tailwind CSSスタイル
- すべてのJavaScriptコード

このファイル1つで完全に動作します！

## 🌐 デプロイ方法

### GitHub Pages（推奨）

1. GitHubリポジトリを作成
2. コードをプッシュ
3. リポジトリ設定で「Pages」→「GitHub Actions」を選択
4. 自動的にデプロイされます

### その他のホスティングサービス

**Netlify:**
```bash
# Netlify CLIをインストール
npm install -g netlify-cli

# デプロイ
cd app
npm run build
netlify deploy --dir=dist --prod
```

**Vercel:**
```bash
# Vercel CLIをインストール
npm install -g vercel

# デプロイ
cd app
npm run build
vercel --prod
```

**手動デプロイ:**
1. `npm run build` を実行
2. `dist/index.html` をWebサーバーにアップロード
3. 完了！

## 🔧 トラブルシューティング

### ポート5173が既に使用されている

```bash
# 別のポートで起動
npm run dev -- --port 3000
```

### ビルドが失敗する

```bash
# node_modulesを削除して再インストール
rm -rf node_modules package-lock.json
npm install
npm run build
```

### データが保存されない

- ブラウザのプライベートモード/シークレットモードを使用していないか確認
- ブラウザの設定でlocalStorageとIndexedDBが有効になっているか確認

### ダークモードが動作しない

1. 設定画面でダークモード切り替えをオン/オフ
2. ブラウザをリロード

## 💡 ヒント

### データのバックアップ
定期的に設定画面から「データをエクスポート」してバックアップを取ることをおすすめします。

### ストリークを維持
毎日トレーニングを完了すると、ストリーク（連続記録）が伸びます。
モチベーション維持に最適！

### プロファイルの変更
運動習慣が変わったら、設定画面から「プロファイルを再設定」で新しいトレーニングプランを生成できます。
